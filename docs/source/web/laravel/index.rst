Laravel
=======

Laravel je framework pre vývoj web aplikácii s kladeným dôrazom na elegantnú syntax a dokumentáciu, ktorá je dostupná na `Laravel dokumentácia <https://laravel.com/docs/9.x>`.

Pre sťahovanie dependencies slúži `Composer <https://getcomposer.org/>` a všetky dostupné balíčky resp. repozitár balíčkov je dostupný na `Packagist <https://packagist.org/>` alebo `GitHub <https://github.com/>`.

Laravel má aj vlastný adresár balíčkov `Packalyst <http://packalyst.com/>` určený pre **Laravel** projekty a teší sa aj veľkému počtu fanúšikov vlastných videotutoriálov `Laracast <https://laracasts.com/>` ako aj `Scotch.io <https://scotch.io/tag/laravel>` tutoriálov.
